# -STM32F407-
本作品设计了一款基于 STM32F407 的仓库物料智能巡检小车，面向中小型仓储环境，实现自动巡检、货架定位、物料计数与缺料告警等功能。系统采用 STM32F407 作为底盘主控，负责小车驱动、PWM 调速、红外循迹、定点停靠以及与 ESP32-S3 的串口通信；ESP32-S3 外接 OV2640 摄像头模块，完成二维码和货架图像采集，并通过 WiFi 将图像上传至 Python Flask 上位机服务。上位机结合二维码识别与 YOLO 目标检测模型，实现货架编号确认、物料自动计数和库存状态判断。检测结果经 ESP32-S3 返回给 STM32F407，用于小车巡检流程控制；当识别到缺料或空货架时，系统可向手机端推送微信告警。
